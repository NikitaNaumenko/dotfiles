### Requirements

* Linux or Mac
* python3
* git
* ansible(python3)
* nvim or vim8 or onivim

### Setup

1. Install dependencies and configs

```sh
$ git clone
$ make install_languages
$ make nvim_configure
$ make install_addons
```

2. Install vim plugins

```sh
$ nvim
:PlugInstall
```

3. Restart nvim
